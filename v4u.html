IronVault Decryptor

Decrypt .ivault backup files from IronVault

Important:
This tool decrypts .ivault files created by IronVault. You need the exact Master Password that was used to create the backup. This tool runs entirely in your browser - no data is sent anywhere.

Decrypted Files

How to Use

  1. Enter your IronVault Master Password
  2. Click "Choose File" and select your .ivault backup file
  3. Click "Decrypt & Extract"
  4. Download each decrypted file individually
Security Note: All decryption happens locally in your browser. Your password never leaves your device.
